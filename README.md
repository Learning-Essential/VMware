# 🌐 VMware Mastery: Installation, Config & Real-World Projects
Welcome to the ultimate VMware Infrastructure repository. This repository is a comprehensive resource designed to take you from a desktop virtualization user to a professional Data Center Administrator.
Inside, you will find step-by-step guides, automated scripts, and real-world deployment scenarios for the entire VMware various tools stack.



## 📁 What’s Inside This Repo?

### 🛠️ 1. Installation & Configuration Guides
Detailed walkthroughs for setting up the core environment:
*   **VMware Workstation Pro:** Setting up your local "Sandbox" and virtual networking.
*   **VMware ESXi (vSphere):** Bare-metal installation, partitioning, and management networking.
*   **vCenter Server (VCSA):** Stage 1 & 2 deployment, SSO domain setup, and inventory organization.

### 🔧 2. VMware Tools & Utilities
Optimization and automation resources for:
*   **VMware Tools / Open-VM-Tools:** Enhancing guest OS performance.
*   **PowerCLI:** PowerShell scripts to automate VM snapshots, host configuration, and health checks.
*   **OVF Tool:** Command-line deployment of virtual appliances.

### 🧪 3. Real-World Scenario Projects
Hands-on projects that mimic production environments:
*   **Project High Availability:** Setting up a 3-node cluster with **vSphere HA and DRS**.
*   **Project Secure Net:** Configuring **Standard & Distributed Switches (vDS)** with VLAN tagging.
*   **Project Storage:** Connecting ESXi to external **iSCSI or NFS** shared storage.

### 📚 4. Tutorials
*   **Networking 101:** Understanding Bridge, NAT, and Host-Only adapters.
*   **Template Management:** How to create "Golden Images" for rapid VM deployment.
*   **Update Manager:** Patching ESXi hosts without downtime using **vSphere Lifecycle Manager**.

### 🚑 5. Troubleshooting Vault
Solutions to common "Gotchas":
*   **Fixing "Host Connection Lost"** in vCenter.
*   **Resolving Snapshot CID chain** errors.
*   **Recovering from Locked Files** or orphaned VMs.
