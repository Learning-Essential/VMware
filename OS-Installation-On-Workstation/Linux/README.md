# VMware Workstation Lab

A collection of VMware Workstation configuration files, automation scripts, and documentation for managing virtualized environments and guest operating systems.

## 🚀 Overview
This repository serves as a central hub for my virtualization projects. It includes optimized configurations for various Linux distributions (specifically Ubuntu) and scripts to streamline the VM lifecycle.

## 📂 Repository Structure
*   **/Configs**: VMX and VMDK configuration templates.
*   **/Scripts**: Shell and PowerShell scripts for automation and maintenance.
*   **/Docs**: Step-by-step guides for OS installation and performance tuning.
*   **/ISO-Links**: Reference links for verified operating system images.

## 🛠️ Environment Details
- **Hypervisor:** VMware Workstation Pro / Player
- **Primary Guest OS:** Ubuntu 24.04 LTS
- **Recommended Base Specs:** 
  - CPU: 2+ Cores
  - RAM: 4GB+ 
  - Disk: 64GB (Thin Provisioned)

## 🔧 Getting Started
1. Clone this repository to your local machine.
2. Follow the guides in the `/Docs` folder to set up your first Ubuntu VM.
3. Use the scripts in `/Scripts` to automate VMware Tools installation.
